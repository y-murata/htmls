新規事業には切っても切れないツールについて　Googleタグマネージャー編

はじめまして。新規事業部の山田です。

私のいるチームは読んで字のごとく、新しいビジネスモデルを生み出すためのチームです。世の中に価値あるサービスが提供できるよう日々、試行錯誤と挑戦を続けています。

そして、この「試行錯誤」という点、当然ですが、やみくもや決め付けでやっていても当然うまく行くものではありません。確実に検証を回していくには定量化が重要になってきます。これはもうどの業界でも当然のことですよね。

特にWEBと呼ばれる世界は、その点は恵まれていて、行動という行動はほとんどデータ化できてしまいます。Analyticsはじめ、誰でも使える便利なツールだって一般的になりはじめていますしね。当然ですが、私たちもそれを十分に活用をしています。計測はじめ出稿からUIに関する指標まで、リソースの少ない新規事業には切っても切れない重要なツールです。

 

そんな中、本日は、Analyticsとセットで活用する「Googleタグマネージャー」を使った基本とも言えるクリックカウントの取得方法についてお話します。

 

何故いまさら？と思われるかもですが、これには経緯もあります。

Goolgeタグマネージャーは今年の6月頃に新UIに完全移行し、古い機能は使えなくなりました。同時にこれまであったリスニング機能が廃止され、新たにトリガーへ機能が統合されています。

その結果、私自身、意外と戸惑ったのと、まだそれほどTipsも検索でヒットしないようなので、共有がてらちょうど良いかな？ということが理由です。

それでは、ご紹介します。

1.プロダクトを選択

左カラムからタグを選択し、新規のボタンをクリック

「名前のないタグ」と出ていますが、ここでは、「test_link_click」としました。タグの名称を決めたら、クリックデータを当然Analyticsに流したいので、GoogleAnalyticsを選択しましょう。

2.タグの種類を選択

ここでは、今お使いのAnalyticsのバージョンで選択してください。最近開設したAnalyticsでしたらほとんどが「ユニバーサル」だと思います。

3.タグを設定

トラッキングIDはGoogleAnalyticsの設定で確認してください。アナリティクスの設定＞プロパティ　で確認できると思います。

今回はクリック履歴をイベントに吐き出します。カテゴリを「click_test」という名称にして、イベントアクションにはクリックされたURLを表示したいので{{Click URL}}を選択しました。

{{Click URL}}は入力スペース横の「変数アイコン（レゴっぽいマーク）」のクリックでドロップダウンしますが、はじめての場合は出てこないので、左カラムの変数をクリックして

クリックの中のClickURLにチェックを入れればOKです。（自動保存されます。）

4.配信するタイミング

今回はクリックを取りたいので当然「クリック」を選択します。 選択すると、何も設定が無ければ以下のページが自動的にモーダルで開きます。 他に設定のある場合はトリガーの一覧がモーダルされますがそこから新規を選択します。

 

今回は外部へのリンククリックのみを取得することするので、名称を「Outbound_click」としました。 ターゲットは「リンクのみ」とします。 なお、すべてにすると、リンク以外の画像などのクリックも取得できます。UI上の誤解などを調査計測したい時に役に立ちます。

次は有効化のタイミングですが、すべての今回はページに適応したいので 正規表現にて「.*」を入力しすべてのページで一致という条件にします。

外部クリックだけを取得したいので、ClickURLに自社サービスのドメインが含まれてない 事を条件にします。サービスのURLという部分に自社のURLを入力してください。 ここまで設定できれば完成です。タグを保存→公開でWEB上に反映させてみましょう。

5.アナリティクスで確認

正常に動いていれば、リアルタイム＞イベント　にて、こんな感じですぐに確認できます。

まとめ

いかがでしてたでしょうか。

Googleタグマネージャー（GTM）はなかなか奥深くて、色々な事ができるツールです。少し手を加えるだけで、無理にSQLを回さずとも必要なデータが取れるようにもなります。

新規事業を通じ、結果はもちろんですが、ちょっとしたプロセスやノウハウについてこれからも発信していけたらと考えています。
