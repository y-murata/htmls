『相手の立場に立って考えてみること』の重要性と難しさ！

はじめまして。ハピタス事業部　パートナーリレーションチームの柴崎です。

パートナーリレーションチームとは、簡潔に説明すると『ハピタス』のコンテンツを作る上で必要不可欠な掲載する広告を取り扱うチームで、日々取引企業様に提案や渉外を行っています。詳しくは、2015.09.28のスタッフブログでマネージャーの高井よりご紹介させていただきましたので、詳しくはそちらをご覧ください。

そのような日々業務に従事する中で、私ができる限り意識しようとしていることをお話しします。

タイトルにも記載したように、それは『相手の立場に立って考えてみること』です。

抽象的な言葉ではよく聞くフレーズで、『具体的にどんなこと！？』と思われると思います。

私の場合ですが、例えば次のようなことがあります。

私は、現在『ハピタス』というWebサービスに携わっていますが、このサービスには顧客として2者いらっしゃいます。1者は、『ハピタス』を利用している、もしくは利用しようとしている消費者の皆様、もう1者は『ハピタス』を通じて事業価値を提供している企業の皆様になります。そして、我々オズビジョンと3者が関わってこのサービスが成り立っています。

日々サービスを提供していく中で、前述の3者の利害が一致しないことが多々あります。

その場合、私は3者それぞれの立場に立って考えてみます。常に出来ているわけではないのですが…

当然、それぞれの立場の視点があるので、その時正しいと思ったことが何かを判断した後、時には企業の皆様に言いづらいことも率直にお伝えしますし、消費者の皆様に対して言いづらいことも、我が社のCSチームを通して忠実にお伝えすることもあります。

また、両者の立場を考えて我々がリスクを背負って対応にあたる場合もあります。

このように、『相手の立場に立って考えてみること』で色んな視点が生まれ、その際より良い判断ができる可能性が広がると思っています。同時に、常にそのような考えを持ちながら業務にあたる難しさも感じていて、それを習慣化することが課題です。

しかしながら、少しは意識できてきたのか一昨年とある取引先様のシンポジウムで個人賞を受賞いたしました。

今後も我々の大事な大事な顧客である消費者の皆様、企業の皆様にお褒めの言葉をたくさんいただけるよう、前進していきたいと思います！
