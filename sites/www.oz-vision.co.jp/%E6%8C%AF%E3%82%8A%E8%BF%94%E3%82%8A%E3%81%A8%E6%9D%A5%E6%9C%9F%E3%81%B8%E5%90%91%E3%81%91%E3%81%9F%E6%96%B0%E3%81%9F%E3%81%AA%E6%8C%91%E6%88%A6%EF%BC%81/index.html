9期目が終わって振り返りと来期へ向けた新たな挑戦！

今日でOZVは9期目が終わります。

今までの9年を振り返ってみると、初めの6年はマネジメントモデル、
残りの3年はビジネスモデルのイノベーションに挑戦してきたと思います。

でどうだったか。まだまだだなと。危機感さえあります。

マネジメントモデルへの挑戦については、理念経営や企業文化にフォーカスし、
人間関係がよく、真面目で、温かい社風になりました。
時には人間関係が最悪な時期もありましたが、
誰かが結婚したり、子供が生まれたら、勝手に誰かが動いて全員でお祝いするし、
本当に嫌な奴が一人もいない。本当に素直で性格がいい人が集まってる感じです。
有給だって誰でも入社1ヶ月目から30日付与されるし、
さらにそれとは別に、毎年自分が大切な人のために特別休暇もとれて、2万円の現金も支給される。
結婚休暇も、産休も、育休も全部あるし、フレックスタイムでリモートワークもOK。
そこにおいては、本当に安心して働きやすい会社だと思います。

でも、それだけでいいんだろうか？
マネジメントモデルへの挑戦の目的は、人の幸せに貢献して自己実現する集団になるため。
そのためには、従来のような経営マネジメントを変えないといけないと思っているから。
でも、顧客やパートナー、社会の幸せにどれだけ貢献できてるのだろうか。
世の中にある様々なサービスの中で、自分たちのサービスはどれほど価値を提供できるのだろうか。
イノベーションと呼べるような挑戦って、どれだけしてきただろう。

ただの良い会社をつくるために、自分は毎日OZVに来ているわけじゃない。
もっと言えば、社員のために会社を作ったんじゃない。
目的を遂げるために、仲間を募ったと言う方が正確だ。
イノベーションを起こして世の中に価値あるサービスを生み出していくことを通じて、
自分の可能性をとことん追求しつづけ具現化させるという幸せを味わう集団をつくるために、
自分は毎日OZVに来ている。

そんな理想の集団なら、もっともっと仕事でワクワクしているだろうし、
無我夢中で新しい価値を世に送り出そうとフロー状態になっているだろうし、
何よりもっと挑戦する人がたくさんいるはずだ。

そんな風に、この9年を振り返って思っている今日この頃でした。
明日からの10期は、わずか3人で始めた頃の様に、何にもないのに楽しくて、
挑戦している会社づくりをしていきたいと思います。
