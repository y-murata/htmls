北洋ビジネスダイレクトご利用環境

北洋ビジネスダイレクトをお申し込みいただく方は、事前に「ご利用環境チェック」を必ず行ってください。

ご利用環境チェック

当行が推奨するビジネスダイレクトのご利用環境は以下の通りです。

画面解像度 XGA（1024×768）以上
通信回線 ブロードバンド（ADSL）以上
ソフトウェア OS（OperatingSystem）やブラウザに応じたMicrosoft WordやワードパッドなどのRTF（リッチテキストフォーマット）対応ソフトをインストールする必要があります。

  • Microsoft Word2010以上をご利用の場合は、Wordが開いたときに「編集を有効にする」を押してください。
  • 一部のソフト（フリーソフトを含みます）では、ご利用いただけません。

ブラウザの制限事項について

【注1】

  • ①北洋ビジネスダイレクトの画面を表示すると、Internet Explorerのアドレスバーが緑色表示されますが、外部ファイル出力によりファイルのダウンロードを行ってからブラウザの戻るボタンや各メニューボタンを押下すると白色表示になる場合があります。

EV　SSL証明書の発行先と発行者が次の内容であることを確認してください。

発行先 bd3.hokuyobank.co.jp
発行者 VeriSign Class 3 Extended Validation SSL CA

発行先と発行者は上記と同じ内容であれば、「EV SSL証明書」は有効に動作しており、ビジネスダイレクトのお取引には影響ありません。なお、緑色表示させるには、一度ログオフを行い、再度ログオンしてください。

  • 「EV　SSL証明書」についてはこちらをクリック
  • 「EV　SSL証明書」の発行先と発行者の確認方法についてはこちらをクリック（PDF156KB）

【注2】

Windows 8 以上で Internet Explorer 10以上をお使いの場合

①Desktopモードのみ対応しています。MetroUIモードには未対応です。

②操作画面内の【戻る】ボタンを押した場合に、2つ以上前の画面に戻る可能性があります。

  • （例）オンライン資金移動時の例では、支払元口座・資金移動先口座・支払金額等を入力後、操作画面内の【戻る】ボタンを押した場合に、前画面ではなく別の画面に移動し、入力した内容が無効となる可能性があります。この場合は、メニューからオンライン資金移動を選択して再度操作を行ってください。

【注3】

電子証明書の取得・更新ができません。

ご利用される場合は、北洋銀行ホームページのURLを「互換表示設定サイト」に登録いただき、互換表示モードにてご利用願います。

  • 操作方法はこちらをクリック（526KB）
  • ※Windows 8.1をご利用の場合で、「電子証明書を生成できませんでした。」と表示されるときは、ログオン後の画面を「信頼済みサイト」に登録した後で操作してください。
    • 操作方法はこちらをクリック（PDF230KB）
  • CATV、イントラネットなど、ネットワーク環境によっては利用できない場合があります（詳細は社内LANのネット ワーク管理者やCATV会社へお問い合わせください）。
  • 海外からの利用については、インターネットに接続して日本語を表示できる環境であればご利用いただける場合がありますが、その国の法律・制度・通信事情・その他の事由により本サービスの利用ができない場合や動作が不安定になる場合がありますので、利用しないでください。

  • ※Netscape CommunicatorはNetscape Communications Corporationの米国およびその他の国における登録商標または商標です。
  • ※Macintosh、Safariは、米国Apple Inc.の米国およびその他の国における登録商標です。
