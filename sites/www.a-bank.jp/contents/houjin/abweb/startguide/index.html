秘密の質問(追加認証)とは

第三者のなりすましによる不正アクセスを防止するセキュリティ対策です。

普段と異なる環境(パソコンやネットワーク)からインターネットバンキングにログインしようとした際、お客さまご本人であることを確認するため「質問」を表示し、お客さまが正しい「回答」を入力することによりご本人を特定させる認証(追加認証)を行う機能です。

 

【登録の例】

質問  当行のお取り引き支店名は？   回答  新町支店
         
質問  会社の筆頭株主は？   回答 青銀太郎

秘密の質問(追加認証)の登録方法

平成26年12月1日(月)以降、ログイン後に以下のような画面が表示されます。

10種類 の質問の中から3つを選択し、各質問について回答を入力してください。

※表示内容は実際の画面と異なる場合があります。

 

【ポイント1】質問は3つ別々のものを選択

【ポイント2】回答は日本語（全角文字）で入力

【ポイント3】登録した質問と回答の内容は忘れずに覚えておく

 

1.追加認証回答登録画面

　質問を選択　→　回答を入力　→「登録」をクリック

 

 

2.追加認証回答登録結果

　登録した内容を確認　→　「確認」をクリック

 

 

3.お手続き完了（トップページが表示されます）

都度指定方式でのお振込み方法の変更について

平成26年12月1日(月)以降、都度指定方式での新規振込先へ振込する場合は、以下の図のように「振込先の管理」メニューで新規登録し、「利用者登録口座一覧から選択」のメニューから振込先を指定する方法に変更となります。

 

実際の新規振込先へ振込手順は以下のとおりです。

手順は大きく２つに分かれます。

まず、 A.振込先登録 を行ってから B.振込依頼 を行います。

くわしい手順を以下にご案内します。

A.振込先登録

A-1.トップページ・資金移動メニュー

　資金移動をクリック　→　振込振替をクリック

 

A-2.振込振替業務選択メニュー・振込先管理作業内容選択メニュー

　「振込先の管理」をクリック　→　「振込先の登録／変更／削除」をクリック

 

A-3.振込先一覧

　「新規登録」をクリック

 

A-4.振込先情報入力

「金融機関名」選択(入力) → 「支店名」選択(入力) → 「口座番号」入力 → 「受取人名」入力→ 「手数料」選択 → 「登録」をクリック

【ポイント1】

登録時は「受取人名」(口座カナ名)の入力が必要です。ただし、振込先が青森銀行分の場合は「受取人名」を省略できます。
【ポイント2】

正確な「受取人名」がわからない場合でも相手先の氏名や会社名などわかる限りの内容を入力して登録してください。

【ポイント3】

 正確な「受取人名」は振込実行時に口座確認機能により自動的に確認し、変更されます(口座確認機能閉塞時を除く）。


 

A-5.振込先登録完了

■引き続き振込振替手続を行う場合 →「トップページへ」をクリック

→ 引き続き「Ｂ.振込ご依頼時のご注意」をご覧ください。

■別の新しい振込先を登録する場合 →「振込先の追加登録へ」をクリック（A-3へ戻る)

■一旦終了する場合 →「トップページへ」をクリック → 「ログアウト」をクリック


 

B.振込ご依頼方法(ご注意)

振込ご依頼時の手順は次のとおりです。

資金移動　→　振込振替　→　新規取引(※1)　→　支払口座選択　→　振込先口座指定方法選択

→　支払金額入力　→　内容確認(※2)

※1：承認機能を利用する場合は「振込データの新規作成」

※2：承認機能を利用する場合は「確定」

 

ご依頼時は以下の２点にご注意ください。

 

B-1.振込先口座の指定方法の選択

振込先口座指定方法選択では「利用者登録口座一覧から選択」をお選びください。

 

B-2.振込振替実行画面での「受取人名」の確認

振込振替の「実行確認」画面では、口座確認機能によって受取人名(口座名義)の確認が行われます。

登録した内容と異なっている場合は赤色で、「ご指定の受取人名と振込先の金融機関に登録されている受取人名が異なる振込先がありましたので、受取人名を変更しました。振込先を十分にご確認の上、取引を行ってください。」のメッセージが表示されますので、変更された内容をご確認の上、振込を実行してください。
