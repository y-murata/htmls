グリルで作るお弁当

レシピID:000176

いかの黄金焼き(1人分74kcal)
五目卵焼き(1人分72kcal・アルミケース2箱分)
えびの塩焼き(1人分13kcal)
焼きおにぎり(1人分226kcal)
手羽先照り焼き(1人分131kcal)
ほっきときのこのみそ焼き(1人分53kcal)
鮭の変わり焼き(1人分105kcal)
材料

    いかの黄金焼き

    • いか 1パイ
    • 卵黄 1個
    • 塩 少々
    • ぎんなん緑8個、黄4個

    五目卵焼き

    • 卵2個
    • だし 大さじ4杯
    • じゃが芋1個
    • 赤ピーマン1/2個
    • 緑ピーマン1/2個
    • ひじき大さじ1杯
    • 塩少々

    えびの塩焼き

    • えび(ブラックタイガー)8尾
    • 塩少々

    焼きおにぎり

    • ごはん50g×8個
    • A調味料
      • みそ 大さじ3杯
      • 酒大さじ2杯
      • みりん大さじ1杯
      • さとう大さじ1杯
      • ごま適量

    手羽先照り焼き

    • 手羽先8本
    • B調味料
      • しょうゆ大さじ4杯
      • 酒大さじ4杯
      • みりん大さじ2杯

    ほっきときのこのみそ焼き

    • ほっき2個
    • しめじ1パック
    • A調味料 焼きおにぎりのみそだれを使います

    鮭の変わり焼き

    • 鮭60g×4切
    • 卵黄1個
    • ホールコーン大さじ2杯
    • 細ねぎ2本
    • 塩少々
    作り方

    いかの黄金焼き

    1. 1. いかは内臓をとり皮をむいて開きます。さらに、半分に切って格子の切り目を入れます。
    2. 2. いかに、げそをのせ塩をふり、巻いてようじでとめ、卵黄をぬります。
    3. 3. グリルにのせ、焼きます。
      　　(予熱2～3分→上下ともに強火で約4分)
    4. 4. ぎんなんはアルミホイルにのせ、グリルで焼きます。　　　(予熱2～3分→上下ともに約4分)

    五目卵焼き

    1. 1. アルミホイルを2枚重ね、縦13×横7×高さ2.5cmの箱をつくります。
    2. 2. じゃが芋は皮をむいて1cm角に切り、さっと茹でて軽くつぶします。
    3. 3. 赤ピーマン、緑ピーマンは5mm角に切ります。ひじきは水でもどしておきます。
    4. 4. ボウルに卵を溶き、だしを加えて塩を入れ2.3.を混ぜて、アルミの箱に流し込みます。
    5. 5. グリルにのせ、焼きます。
      　　(予熱2～3分→上下ともに中火で約7分)

    えびの塩焼き

    1. 1. えびは皮をむき背わたを取って塩をふります。
    2. 2. グリルにのせ、焼きます。
      　　(予熱2～3分→上下ともに強火で約3分)

    焼きおにぎり

    1. 1. おにぎりを握ります。
    2. 2. 鍋にA調味料を入れ煮詰めます。
    3. 3. おにぎりに2.のみそだれをぬり、ごまをふります。
    4. 4. グリルにのせ、焼きます。
      　　(予熱2～3分→上下ともに強火で約5分)

    手羽先照り焼き

    1. 1. 手羽先に切り込みを入れ、B調味料に15分程度漬け込みます。
    2. 2. グリルにのせ、焼きます。
      　　(予熱2～3分→上弱火、下強火で約6分)

    ほっきときのこのみそ焼き

    1. 1. ほっきは開いて内臓を取り、1cm幅くらいに切ります。
    2. 2. しめじは小房に分けておきます。
    3. 3. アルミホイルに別々にのせ、みそだれをぬります。
    4. 4. グリルにのせ、焼きます。
      　　(予熱2～3分→上下とも強火で、
      　　　　　　しめじは約3分、ほっきは約5分)

    鮭の変わり焼き

    1. 1. 鮭は塩をふり、両褄(りょうづま)折りに(皮を表にして、両端を内側に折り曲げて)串を打ちます。
    2. 2. ねぎは小口切りにします。
    3. 3. 鮭に卵黄を塗り、ホールコーンをのせてねぎをふります。
    4. 4. グリルにのせ、焼きます。
      　　(予熱2～3分→上下とも中火で約4分)

    カップ1杯=200ml 大さじ1杯=15ml 小さじ1杯=5ml

    両面焼きグリルは上下別々に温度調節ができ、短時間で仕上がります。また、片面水ありタイプのグリルやオーブンでもつくることができますので、加熱時間を調節して試してください。
