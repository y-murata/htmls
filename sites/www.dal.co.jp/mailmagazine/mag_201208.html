■ACMS E2X、ACMS B2Bの最新版3.9.1をリリース(2012/7/31)
　<トピックス>
　・災害対策用としてのホスト名変更コマンドの提供
　・運用画面の脆弱性への対応
　・運用者認証失敗時のロックアウト・メッセージの改善
　・メールEDIにおけるPOP3受信時のUIDLサイズの拡張
　・スケジュール情報抽出時のパフォーマンス改善と抽出開始終了ログ出力
　・集配信情報のクリーンアップ処理終了時間指定機能の追加
　・FAX 連携における送信先情報の登録作業の簡素化
　・Web-EDI機能「ACMS Web/deTrade II」の自動送受信コマンド障害時のリト
　　ライ機能追加
　・トランスレータ AnyTran 5.15.1を実装

  <動作環境対応>
   [DB] SQL Server 2012、PostgreSQL 9.1(*1)
   [JDBC] JDBC Driver 4.0 for SQL Server(*1)
   [WAS] Tomcat 6.0.35（CD同梱）
         Interstage Application Server 10.0.0(*1)

   (*1)PostgreSQL 9.1、JDBC Driver 4.0 for SQL Server、
       Interstage Application Server 10.0.0は、ACMS E2X、ACMS B2Bでの
　　　 みサポートしています。

　↓詳しくはこちら↓
　【プレスリリース】http://www.dal.co.jp/pressrel/20120727.html
　【ACMS E2X】http://www.dal.co.jp/products/edi/e2x/outline.html
  【ACMS B2B】http://www.dal.co.jp/products/edi/b2b/outline.html
