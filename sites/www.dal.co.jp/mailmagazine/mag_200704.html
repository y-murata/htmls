　　EDI最前線／
　　電機・電子部品業界調達業務向けWeb-EDI「BUYSELWARE」導入事例
----------------------------------------------------------------------

安川シーメンスオートメーション・ドライブ株式会社様

「新Web-EDIシステムに『BUYSELWARE』を採用
調達・ロジスティック業務の質の向上、そして戦略化を実現」

エンジニアリング会社も含め製造業が抱える課題の一つとして、調達業務の質
の向上と戦略化が挙げられる。産業用オートメーション＆ドライブのソリュー
ションプロバイダーでリーディングカンパニーである安川シーメンスオートメ
ーション・ドライブ株式会社（以下YSAD）でも、人手と紙ベースに頼る部分が
残るプロセスはもとより、サプライヤー毎に微妙に違うワークフローをいかに
効率化し、連携を深めるかが重要なテーマとなっていた。

そこで同社が新たに導入したのが、DALが提供するWeb-EDIシステム「BUYSELWA
RE」だ。見積・発注から検収、請求に至るまでのプロセスを、トータルに最適
化することに成功。また次世代EDIへの対応や、調達・ロジスティックの戦略
構築にも役立てている。

　　本文⇒ http://www.dal.co.jp/casestudies/12scs/index.html

----------------------------------------------------------------------
　　パートナーインタビュー／ 住商情報システム株式会社様
----------------------------------------------------------------------
パートナーインタビューでは、弊社販売パートナー様をご紹介します。
第3回目は、住商情報システム株式会社（以下SCS） 常務執行役員 栗本氏 と
弊社 代表取締役社長 橋本との対談となりました。　

「EDIから、EAI (企業内連携) やESB (エンタープライズ・サービス・バス) まで。
シームレスなシステム連携により、お客様の視点に立った経営支援を目指す。」
住商情報システム株式会社
常務執行役員　栗本 重夫氏

2005年8月の合併時点は社内カンパニー制を導入していたため、合併したとい
っても、実際は、それぞれに活動していたというのが実情です。その後、昨年
4月に改めて住商情報システムとして各々を整理、統合し、現在は、12の事業
部に分かれて活動を展開しています。

その中で、DALさんのACMSを扱う「IT基盤ソリューション事業部」は、インダ
ストリー別の「縦串」事業部に対して、全社の「横串」機能として、システム
基盤、セキュリティ、ネットワークなどのIT基盤を提供しています。

システム基盤では、サーバ基盤、運用基盤など、さまざまなサービスを提供し
ていますが、ACMSは、その中核を成す製品のひとつとして位置づけられていま
す。

　　本文⇒ http://www.dal.co.jp/mailmagazine/interview_0704.html

　　多目的データ変換ツール、
　　「AnyTran サーバ版」の最新バージョン5.5をリリース
----------------------------------------------------------------------
Microsoft Windows Vista対応、「AnyTranサーバ版」の最新バージョン5.5を、
2007年3月30日にリリースしました。

AnyTranは、プログラミングなしでデータ変換やフォーマット変換、漢字コー
ド変換などを可能にする多目的データ変換ツールです。
これにより、企業電子取引や企業内外のデータ連携をスムーズに行い、EDI、
EAIおよびBPM構築におけるシステム開発・管理工数を大幅に削減します。

■主な新機能は次のとおりです。
　　1. Microsoft Windows Vistaへの対応
　　Microsoft Windows Vista上でAnyMapperによる定義の作成およびフォーマット
　　変換が可能です。

　　2.文字コードJIS X 0213:2004への対応
　　JIS X 0123:2004文字コードをサポートしました。
　　これにより、Window Vistaの文字コードセット、UNICEODEをはじめ、シフトJI
　　SやEUCコード、JISコード間でのX0213文字コード変換も可能になります。

　　3.XMLスキーマによる妥当性検証機能の追加
　　入力／出力がXMLフォーマットの場合、XMLスキーマを利用して、XMLデータの
　　妥当性検証ができます。これにより、事前に入出力のXMLデータをチェックで
　　きるので、整合性のあった正しい変換データの作成が可能になります。

その他、製品情報は以下をご覧ください。
