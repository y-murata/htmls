 ■ACMS E2X、ACMS B2B最新バージョン3.4.0をリリース（2009/07/31）


今回のリリースでは、両製品共に、電子情報技術産業協会のJEITA/ECALGAで
採用されている企業間電子商取引の国際標準であるebXML MS 3.0のサーバ機能
と、石油化学業界の企業間電子商取引の国際標準であるChem eStandards v4.0
を新たに実装しております。また、その他の主な新機能は、以下のとおりです。
詳細は、プレスリリースおよびリリースノートをご覧ください。

[主な新機能]
 1. ebXML MS 3.0準拠のJEITA共通クライアントVer1.0に対応したサーバ機能
　 を実装 
 2. CEDIガイドラインのChem eStandards v4.0を実装 
 3. Microsoft SQL Server 2008への対応 
 4. 以下のJDBCドライバとサポート 
  　Microsoft SQL Server 2005 JDBC Driver 1.2 
  　Microsoft SQL Server JDBC Driver 2.0 
 5. ACMS管理画面およびACMS Web/deTrade IIにおいてMicrosoft Internet 
　　Explorer 8をサポート 
 6. UNISYS漢字とその他の文字コード間で文字コード変換

是非、この機会にご参加ください。

・メインフレームでEDIを運用しているが維持管理や運用コストを改善したい。
・VANやASPの利用で、サービス内容や料金に対して不満がある。
・Web-EDＩや流通BMSの対応で困っている。

■日立SAS、マイクロソフト、DAL共催セミナー（大阪）
 
 「流通業における受発注と経営全体の効率化」
 　〜 EDIの最新動向とERPパッケージ〜

現在の流通・サービス業全体においては、スケールメリットを生かしたディス
カウント志向と、店舗や品揃えの独自性をアピールする差別化志向の二極化の
方向といわれております。いずれも実現するためには、直接的原価・販売管理
費用の低減が重要な鍵となります。この鍵となるのが受発注業務・経営全体の
効率化、すなわち次世代EDIと統合業務システム（ERP）といえるのではないで
しょうか。
そこで今回は、EDIから業務システム連携製品のトップメーカであるデータ・
アプリケーション社のACMS製品とデータの連携が標準装備されているERPパッ
ケージ"Microsoft Dynamics AX2009"のデモンストレーションを交え、分か
りやすくご紹介いたします。

 【日時】2009/09/04（金）14:00〜16:15（受付開始 13:30）
 【会場】なんばパークスタワー　19Ｆ セミナールーム
　　　　 大阪市浪速区難波中2-10-70　パークスタワー19F
 【共催】株式会社日立システムアンドサービス
　　　　 マイクロソフト株式会社
　　　　 株式会社データ・アプリケーション
 【費用】無料
