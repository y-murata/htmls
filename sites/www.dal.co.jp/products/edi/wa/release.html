EDI（企業間電子商取引）

様々な形態のB2B（企業間電子取引)を成功に導くミドルウェア製品を提供しています。

煩雑になったWeb-EDIのブラウザ操作を自動化することで業務効率の向上を実現

  • ACMS WebAgentの最新バージョン2.2.1を2014/5/12にリリースしました。

 

ACMS WebAgent｜ACMS WebAgent LE｜Script Studio  
ACMS WebAgent Version 2.2.3 リリース情報　2015/5/29

■機能改善

  1. COM関連のエラーコード出力
    巡回内でのCOM関連で障害が発生した場合のエラーコードを出力するようにしました。

■解決された問題点

[コア機能]

  1. Log.write()にて改行コードが変換されない問題
    ユーザ用ファイル出力の設定がチェックなし（SJIS指定）の場合で、Log.write()にて改行コード<LF>の出力を行なうとうと<CRLF>に変換されない問題に対応しました。

  2. 不正な後処理プログラムを指定した場合に重複受信してしまう問題
    異常終了時の後処理プログラムの指定で不正な値を指定しエラーになった場合、既に受信済みのファイルが正常終了時の後処理プログラムで重複受信してしまう問題に対応しました。

  3. フォルダ作成時に障害が発生してしまう問題
    KeyAgent.createDirectory()にて既に存在するフォルダに対し作成処理を行なうと、重複エラーが発生しエラーコード(-1214)で異常終了してしまう問題に対応しました。

ACMS WebAgent Version 2.2.0　リリース情報　2013/11/29

■トピックス

  1. IE起動失敗時のリトライ機能
    巡回時のIE起動エラー時にリトライを３回行うようになりました。

■機能改善

  1. IE起動失敗時のリトライ機能
    巡回時のIE起動（DCOM 経由でリモートのCOM 呼び出し）の際に、
    0x800706ba (RPC_S_SERVER_UNAVAILABLE) エラーが発生した場合、
    リトライを３回行うようになりました。

ACMS WebAgent Version 2.1.0　リリース情報　2012/7/31

■トピックス

  1. 中国語（簡体字）のサポート
    中国語（簡体字）サイトの巡回をサポートしました。

  2. マウス操作、キーボード操作、文字列送信の関数が追加されました。
    IEの上部メニューの操作、右クリックメニューの操作、JavaアプレットやFlashなどの
    HTML以外の言語で記述された表示領域への操作が一部可能となります。

  3. マウス操作を行う為の画面最大化用関数が追加されました。

  4. Microsoft Windows Server 2008 R2 SP1 に対応しました。

  5. Internet Explorer 8 に対応しました。

■新機能

  1. 中国語（簡体字）のサポート
    中国語（簡体字）サイトの巡回ができるようになりました。
    取引先単位にファイル出力およびログファイルをUTF-16で出力可能になります。

  2. CTRL オブジェクトの追加

    1) mouseClick
    デスクトップ上の座標指定した位置に対してマウス操作（右ボタンクリック、
    左ボタンクリック）ができるようになりました。
    2) sendKey
    キー操作相当の文字またはコードを指定することにより、キーボード操作が
    できるようになりました。
    3) sendString
    指定した文字列を送信することができるようになりました。

  3. IE オブジェクトの追加

    1) setWindow
    画面の最大化ができるようになりました。

  4. 下記のWindowsOSとサービスパックに対応しました。

    Windows Server 2008 R2 Standard Edition SP1
    Windows Server 2008 R2 Enterprise Edition SP1

  5. 下記のWebブラウザに対応しました。

    Internet Explorer 8

■機能改善

    IE.setInputでbuttonタイプの押下処理を行う際に、ボタンに対して直接フォーカスを当て、
    押下処理を行うようにしました。
